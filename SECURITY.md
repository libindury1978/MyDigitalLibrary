# 安全策略

## 支持的版本

| 版本   | 支持状态 |
|--------|----------|
| 最新 Release | ✅ 优先修复 |
| `main` 分支最新提交 | ✅ 尽力修复 |
| 更旧版本 | ❌ 不保证 |

## 报告漏洞

**请勿在公开 GitHub Issue 中报告可被利用的安全漏洞。**

请通过以下方式私下联系维护者：

- GitHub：[libindury1978](https://github.com/libindury1978)（Security Advisory 或私信，若已启用）
- 或在仓库 Settings 中配置的 Security contact（如有）

报告时请尽量包含：

1. 漏洞类型与影响（例如：路径遍历、任意文件读写）
2. 受影响版本
3. 复现步骤或 PoC（最小化即可）
4. 可能的缓解措施（若已知）

我们会在合理时间内确认收到，并在修复可用后协调公开披露时间。

## 范围说明

本应用为本地桌面程序，主要风险包括：

- 对用户所选目录/文件的路径处理不当
- Tauri 能力（capabilities）配置过宽
- 通过 `asset` 协议暴露本地资源

不属于典型 Web XSS 场景，但仍请报告任何可导致未授权访问用户文件或远程代码执行的问题。
